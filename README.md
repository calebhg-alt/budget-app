# Ledger — Personal Budget & Meal Prep

A single-file personal finance app: transactions, category budgets, savings goals, reports, and a meal-prep planner that estimates weekly grocery cost against a budget.

No build step, no backend — everything runs client-side and saves to your browser's localStorage.

## Live app
https://calebhg-alt.github.io/budget-app

## Data
All data (transactions, budgets, goals, recipes, meal plan) is stored in your browser's localStorage. Use the **Export backup** button in the sidebar regularly, since localStorage is per-browser and can be cleared.
